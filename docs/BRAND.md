# DAIRA Brand Guidelines

## Name & Meaning

**DAIRA** (دائرة) is the Arabic word for "circle," representing:
- Community and connection
- Circular conversations and discussions
- Expanding one's social circle
- The cyclical nature of content and engagement

## Tagline

**"Share your world. Expand your circle."**

This encapsulates our mission: enable users to share their experiences while growing their community.

## Color Palette

### Primary Colors

| Color | Hex | Usage |
|-------|-----|-------|
| **Nile Blue** | `#0D2C56` | Primary brand color, CTAs, headers, links |
| **Sandstone** | `#D9C8A0` | Borders, dividers, secondary accents |
| **Basalt** | `#2C2C2C` | Body text, dark backgrounds |
| **Electric Mint** | `#2CF5C4` | Highlights, active states, success messages |
| **Off-white** | `#FAFAF8` | Page backgrounds, light surfaces |

### Usage Guidelines

- **Nile Blue**: Use for primary actions, navigation, and branding
- **Electric Mint**: Use sparingly for energy and emphasis
- **Sandstone**: Use for subtle separation and warmth
- **Basalt**: Use for readable text on light backgrounds
- **Off-white**: Use for backgrounds to reduce eye strain

### Accessibility

All color combinations meet WCAG 2.1 AA standards:
- Nile Blue on Off-white: 10.2:1 (AAA)
- Basalt on Off-white: 13.1:1 (AAA)
- Electric Mint on Basalt: 8.4:1 (AAA)

## Typography

### Primary Font: Inter

**Inter** is used for all English content:
- Clean, modern sans-serif
- Excellent readability at all sizes
- Wide range of weights (400, 500, 600, 700)
- Open-source and web-optimized

### Secondary Font: Cairo

**Cairo** is used for Arabic content:
- Designed specifically for Arabic script
- Maintains consistency with Inter's aesthetic
- Supports RTL (right-to-left) layout
- Weights: 400, 500, 600, 700

### Type Scale

```
Heading 1: 48px / 3rem (Bold)
Heading 2: 36px / 2.25rem (Bold)
Heading 3: 28px / 1.75rem (Semibold)
Heading 4: 24px / 1.5rem (Semibold)
Body Large: 18px / 1.125rem (Regular)
Body: 16px / 1rem (Regular)
Small: 14px / 0.875rem (Regular)
Caption: 12px / 0.75rem (Medium)
```

## Logo Usage

### Logo Variants

1. **Full Logo**: DAIRA with icon (primary)
2. **Icon Only**: Circle symbol (small spaces)
3. **Wordmark**: DAIRA text only (limited space)

### Clear Space

Maintain minimum clear space of 2x the height of the logo on all sides.

### Minimum Size

- Full logo: 120px wide
- Icon only: 32px
- Wordmark: 80px wide

## Microcopy

### Voice & Tone

- **Friendly**: Warm and approachable
- **Inclusive**: Welcoming to all cultures
- **Clear**: No jargon, simple language
- **Encouraging**: Positive and supportive

### Key Phrases

- "Share your world"
- "Expand your circle"
- "Join the conversation"
- "Your story matters"
- "Connect authentically"

### Button Labels

- Primary action: "Post", "Share", "Join"
- Secondary action: "Cancel", "Skip", "Maybe Later"
- Destructive action: "Delete", "Remove", "Block"

### Error Messages

- Friendly: "Oops! Something went wrong."
- Actionable: "Let's try that again."
- Helpful: "Check your connection and retry."

## Iconography

### Style

- Rounded corners (2px radius)
- 2px stroke weight
- 24px grid system
- Consistent optical balance

### Icon Set

Use consistent icons from one library:
- Heroicons (recommended)
- Feather Icons (alternative)
- Custom icons following the same style

## Imagery

### Photography Style

- **Authentic**: Real people, real moments
- **Diverse**: Representing all communities
- **Vibrant**: Energetic but not oversaturated
- **Composed**: Well-framed with clear subjects

### Illustrations

- **Minimal**: Simple, clean line work
- **Geometric**: Circular motifs when possible
- **Colorful**: Using brand palette
- **Inclusive**: Diverse representation

## Motion & Animation

### Principles

- **Purposeful**: Every animation serves a function
- **Subtle**: Smooth, non-distracting
- **Quick**: 150-300ms for most transitions
- **Natural**: Easing curves (ease-in-out)

### Common Transitions

- Page transitions: 200ms fade
- Button hover: 150ms scale + color
- Modal open: 250ms slide + fade
- Loading states: Skeleton screens

## Responsive Design

### Breakpoints

```
Mobile: < 640px
Tablet: 640px - 1024px
Desktop: > 1024px
```

### Layout Principles

- Mobile-first approach
- Fluid typography
- Touch-friendly targets (44px minimum)
- Simplified navigation on mobile

## Accessibility

### Requirements

- WCAG 2.1 AA compliance minimum
- Keyboard navigation support
- Screen reader compatibility
- High contrast mode support
- Focus indicators on all interactive elements

### Testing

- Test with screen readers (NVDA, JAWS, VoiceOver)
- Keyboard-only navigation testing
- Color blindness simulation
- Mobile accessibility testing

## Localization

### RTL Support

Full right-to-left layout support for Arabic:
- Mirrored layouts
- Text alignment
- Icon direction
- Navigation flow

### Cultural Sensitivity

- Avoid idioms that don't translate
- Use culturally neutral imagery
- Respect cultural holidays and events
- Inclusive language for all regions
